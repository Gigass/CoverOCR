# 📘 机器学习书本封面识别项目需求说明文档（V1.0）

## 一、项目目标（核心任务）

构建一个**基于机器学习的图像识别系统**，能够：

1. **识别书本封面图像**（教材、参考书等）；
2. **提取封面上的文字信息**，包括：
   - **中文文字**
   - **英文文字**
   - **字体样式（字体类型）**
3. **支持用户上传实拍照片**，系统自动识别并输出识别结果；
4. **最终验收方式**：用户现场拍摄一本书的封面，上传后系统完成识别并返回识别结果。

---

## 二、功能需求

### 2.1 图像识别模块

| 功能点 | 描述 |
|--------|------|
| 封面识别 | 输入一张书本封面图像，系统识别出这是哪本书（或是否为目标书） |
| 文字识别 | 识别封面上的所有可见文字，包括中文、英文 |
| 字体识别 | 识别封面中文字的字体样式（如宋体、黑体、Arial、Times New Roman 等） |
| 图像输入支持 | 支持用户上传 JPG/PNG 格式图像 |
| 图像来源兼容性 | 支持用户**实拍图像**（非扫描、非标准角度、非均匀光照） |

---

### 2.2 用户交互模块（验收用）

| 功能点 | 描述 |
|--------|------|
| 图像上传 | 用户可通过界面上传一张实拍的书本封面图像 |
| 识别结果展示 | 系统返回以下识别结果：<br> - 是否识别为某本已知书（或书名）<br> - 封面中提取的中英文文字内容<br> - 识别出的字体类型 |
| 响应时间 | 上传后识别结果应在**合理时间内返回**（建议 ≤ 10 秒） |

---

## 三、性能需求

| 指标 | 要求 |
|------|------|
| 封面识别准确率 | ≥ 85%（在测试集上） |
| 中英文文字识别准确率 | ≥ 80%（文字内容正确） |
| 字体识别准确率 | ≥ 70%（字体类别正确） |
| 实拍图像鲁棒性 | 支持轻微角度倾斜、光照不均、背景干扰等情况下的识别 |
| 支持图像分辨率 | 最低支持 640×480，建议支持 1920×1080 以内 |

---

## 四、数据需求

### 4.1 图像数据

| 类型 | 描述 |
|------|------|
| 封面图像 | 包含教材、参考书等典型书本封面，来源包括网络、教材扫描、实拍等 |
| 图像数量 | 每本书至少 5 张不同版本/角度/质量的图像（用于训练和测试） |
| 图像多样性 | 包括不同出版社、不同版本、不同拍摄条件下的封面图像 |
| 图像标注 | 每张图像需标注：<br> - 书名（中英文）<br> - 封面文字内容（中英文）<br> - 字体样式（如可识别） |

---

### 4.2 文字与字体数据

| 类型 | 描述 |
|------|------|
| 中英文字体样本 | 收集常见字体（如宋体、仿宋、黑体、楷体、Arial、Times 等）的文字图像样本 |
| 字体标注 | 每个文字图像需标注其字体类型 |
| 文字区域标注 | 封面图像中需标注文字区域（可用于训练文字检测模型） |

---

## 五、模型需求（非代码）

| 模型类型 | 用途 |
|----------|------|
| 图像分类模型 | 用于识别封面属于哪本书 |
| 文字检测模型 | 用于定位封面中的文字区域 |
| 文字识别模型 | 用于识别中英文文字内容 |
| 字体分类模型 | 用于识别文字的字体样式 |

---

## 六、验收标准（最终交付依据）

### 6.1 验收方式

- **现场拍摄**：验收人员随机选择一本教材/参考书，现场拍摄封面；
- **上传识别**：用户将照片上传至系统；
- **输出结果**：系统返回以下信息：
  1. 是否识别出书名（或匹配到数据库中的书记录）；
  2. 封面中提取的中英文文字内容（逐条列出）；
  3. 识别出的字体类型（如识别出“标题为黑体，正文为宋体”）；
- **人工核对**：验收人员人工核对识别结果是否正确。

---

### 6.2 验收通过标准

| 项目 | 要求 |
|------|------|
| 封面识别 | 正确识别出书名或匹配到正确记录 |
| 文字识别 | 封面中主要文字（书名、作者、出版社等）识别正确率 ≥ 80% |
| 字体识别 | 能识别出至少一种主要字体（如标题字体） |
| 实拍图像支持 | 支持现场实拍图像，非标准扫描图像 |
| 系统响应 | 上传后 10 秒内返回识别结果 |

---

## 七、非功能需求

| 类别 | 要求 |
|------|------|
| 可扩展性 | 系统应支持后续添加新书、新字体、新语言 |
| 易用性 | 用户界面简洁，支持一键上传与结果查看 |
| 兼容性 | 支持主流浏览器、手机端上传图像 |
| 安全性 | 上传图像仅用于识别，不存储用户隐私数据 |

---

## 八、项目限制与假设

| 项目 | 描述 |
|------|------|
| 图像质量 | 假设用户上传图像清晰，文字可见，非极端模糊 |
| 字体库范围 | 初期仅支持识别常见中文字体（如宋体、黑体、楷体、仿宋）和英文字体（如 Arial、Times、Helvetica） |
| 封面类型 | 初期以教材、参考书为主，暂不支持杂志、报纸、非标准封面 |
| 语言支持 | 初期支持中文、英文，暂不支持其他语言 |
