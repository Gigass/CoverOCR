# CoverOCR 新手启动教程（Windows 版）

本教程专为 Windows 用户编写，手把手教您如何从命令行（CMD）启动 CoverOCR 项目。

---

## �️ 第零步：环境准备（必做！）

在开始之前，我们需要先安装两个必要的软件：**Python** 和 **Node.js**。
如果您的电脑上已经安装过，可以跳过这一步。

### 1. 安装 Python (后端运行环境)

1. **下载**：
   - 访问官网下载页面：[https://www.python.org/downloads/](https://www.python.org/downloads/)
   - 点击黄色的 **"Download Python 3.x.x"** 按钮下载安装包。

2. **安装（⚠️ 关键一步）**：
   - 双击运行下载的安装包。
   - **【非常重要】** 在安装界面的底部，**务必勾选 "Add Python.exe to PATH"**（将 Python 添加到环境变量）。
     > 如果不勾选这个，后面在 CMD 里输入 python 会提示找不到命令！
   - 然后点击 **"Install Now"** 进行安装。

3. **验证**：
   - 打开 CMD（按 Win+R，输入 `cmd`，回车）。
   - 输入 `python --version` 并回车。
   - 如果显示 `Python 3.x.x`，说明安装成功！

### 2. 安装 Node.js (前端运行环境)

1. **下载**：
   - 访问官网：[https://nodejs.org/](https://nodejs.org/)
   - 点击左边的 **"LTS"** 版本（长期支持版）按钮下载。

2. **安装**：
   - 双击运行安装包。
   - 一路点击 **"Next"**（下一步）直到完成即可，默认设置通常没问题。

3. **验证**：
   - 打开 CMD。
   - 输入 `node -v` 并回车。
   - 如果显示 `v16.x.x` 或更高的版本号，说明安装成功！

4. **配置加速（强烈推荐！🇨🇳）**：
   - 因为国内网络原因，直接下载可能会失败或卡住。
   - 请在 CMD 中复制并运行以下命令，使用国内镜像源：
     ```cmd
     npm config set registry https://registry.npmmirror.com
     ```
   - 这样后续下载速度会快很多，也能避免报错。

## 📍 第一步：打开命令提示符（CMD）

### 方法一：通过搜索打开（推荐）
1. 按键盘上的 **Win 键**（键盘左下角那个 Windows 图标）
2. 直接输入字母：`cmd`
3. 在搜索结果中点击 **"命令提示符"** 或 **"Command Prompt"**

### 方法二：通过运行窗口打开
1. 按 **Win + R** 组合键
2. 在弹出的框中输入 `cmd`
3. 点击 **确定**

### 窗口长什么样？
打开后，您会看到一个黑色的窗口，里面有类似这样的文字：
```
C:\Users\YourName>
```

---

## 📂 第二步：进入项目目录

### 1. 获取项目路径
首先，您需要知道项目文件夹在哪里。
1. 打开文件夹（资源管理器），找到 `CoverOCR` 文件夹。
2. **按住 Shift 键**，同时**右键点击** `CoverOCR` 文件夹。
3. 在菜单中选择 **"复制文件地址"** 或 **"Copy as path"**。

### 2. 在 CMD 中进入目录
回到刚才打开的黑色 CMD 窗口，输入以下命令（注意 `cd` 后面有个空格）：

```cmd
cd /d 
```

然后**右键点击** CMD 窗口内部（或者按 Ctrl+V），粘贴刚才复制的路径。
最终看起来应该像这样（注意路径两边可能有引号，没关系）：

```cmd
cd /d "D:\Projects\CoverOCR"
```

**💡 为什么要加 `/d`？**
如果您的项目在 D 盘，而 CMD 默认在 C 盘，不加 `/d` 是进不去的。加上它最保险！

按 **回车键** 执行。

### 3. 确认是否成功
输入 `dir` 命令并回车：
```cmd
dir
```
如果您能看到 `start_local.bat`、`backend`、`frontend` 等文件，说明成功了！✅

---

## 🚀 第三步：分别启动后端和前端

为了更稳定地运行项目，我们提供了两个独立的启动脚本。请**分别双击运行**它们。

### 1. 启动后端 (Backend)

找到项目目录下的 **`start_backend.bat`** 文件，双击运行。

*   脚本会自动配置环境并启动服务。
*   看到 `Application startup complete` 说明后端启动成功。
*   **请不要关闭这个窗口！**

### 2. 启动前端 (Frontend)

找到项目目录下的 **`start_frontend.bat`** 文件，双击运行。

*   脚本会自动安装依赖并启动服务。
*   看到 `Local: http://localhost:5173/` 说明前端启动成功。
*   **请不要关闭这个窗口！**

## 🌐 第四步：访问应用

打开您的浏览器（Chrome/Edge），访问：

👉 **http://localhost:5173**

您现在应该能看到 CoverOCR 的界面了！🎉

---

## 🛑 第五步：停止项目

当您不想用了，可以通过以下方式停止：

### 直接关闭窗口
直接关闭那两个黑色的 CMD 窗口（Backend 和 Frontend），服务就会自动停止。

## ❓ 常见问题

### Q1: 提示 `'python' 不是内部或外部命令`？
**原因**：您没有安装 Python 或者没有添加到环境变量。
**解决**：
1. 下载安装 [Python 3.9+](https://www.python.org/downloads/)。
2. **关键**：安装时一定要勾选 **"Add Python to PATH"**（将 Python 添加到环境变量）。

### Q2: 提示 `'npm' 不是内部或外部命令`？
**原因**：您没有安装 Node.js。
**解决**：
1. 下载安装 [Node.js](https://nodejs.org/) (推荐 LTS 版本)。
2. 安装完成后重启 CMD 窗口。

### Q3: 如何清除缓存/重新安装依赖？
**场景**：如果遇到依赖冲突、版本问题，或想彻底重装环境。

**清除后端缓存**：
1. **先关闭后端服务**（关闭运行 `start_backend.bat` 的窗口）。
2. 删除项目根目录下的 `.venv` 文件夹。
3. 重新运行 `start_backend.bat`，脚本会自动重建环境。

**清除前端缓存**：
1. **先关闭前端服务**（关闭运行 `start_frontend.bat` 的窗口）。
2. 删除 `frontend` 目录下的 `node_modules` 文件夹。
3. 重新运行 `start_frontend.bat`，脚本会自动重新安装依赖。

**注意**：删除后首次启动会比较慢，因为需要重新下载所有依赖包。

2.  双击 `start_frontend.bat` 启动前端。
3.  浏览器访问 `http://localhost:5173`。
